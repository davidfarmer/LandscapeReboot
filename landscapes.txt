# landscapes.txt -- registry of L-function landscapes.
#
# Each [landscape NAME] block fully specifies a searchable family of tempered
# balanced analytic L-functions (arXiv:1711.10375, section 2.1).  The same file
# format is used for an ad-hoc landscape passed on the command line with
# --landscape-file FILE (for development); the named registry below is the
# production list loaded by --landscape NAME.
#
# Format
# ------
#   [landscape NAME]            start a block; NAME is the registry key
#   key = value                 one per line; '#' starts a comment
#   gamma = ...                 may appear several times (one Gamma factor each)
#
# Keys
#   description  free text
#   degree       d   (must equal d1 + 2*d2 from the signature)
#   signature    d1, d2   -- d1 Gamma_R factors and d2 Gamma_C factors
#   conductor    N   (default 1)
#   character    Dirichlet character mod N.  'trivial' (default) or a Conrey
#                label q.n.  Fixes the good-prime top coefficient (Ax5a:
#                F_p(z) = 1 - a_p z + ... + (-1)^d chi(p) z^d) and, with the
#                Gamma shifts, the parity (Ax5c).
#   self_dual    true | false (default false).  When true the L-function is
#                self-dual and the good-prime local factor is self-reciprocal
#                (real coefficients, roots closed under z -> chi(p)/z),
#                c_{d-k} = (-1)^d chi(p) c_k.  When false the local factor is only
#                TWISTED self-reciprocal (the relation carries a conjugation,
#                c_{d-k} = (-1)^d chi(p) conj(c_k)), as for a generic GL(3) Maass
#                form, whose dual is the contragredient.
#   gamma        one Gamma factor, written
#                    R(delta | c1, c2, ...)      for Gamma_R(s + delta + i*(c.lambda))
#                    C(kappa | c1, c2, ...)      for Gamma_C(s + kappa + i*(c.lambda))
#                delta in {0,1}, kappa in {1/2,1,3/2,...} are the Selberg real
#                shifts (Ax4a), constrained by the character parity.  The
#                coefficients c1,c2,... give the imaginary part as a combination
#                of the free spectral parameters lambda1..lambda_r; they must
#                make the family balanced, Im(sum mu + sum(2 nu + 1)) = 0 (Ax5b).
#                The number of free spectral parameters r (= the search
#                dimension) is the length of these coefficient lists.
#                Either way the Satake roots lie on |z|=1 with product chi(p), so
#                the local factor is fixed by its first floor((d-1)/2) Dirichlet
#                coefficients b(p), b(p^2), ...  (degree 3,4 -> 1 unknown; 5,6 -> 2).
#
# Bad primes (p | N) are not yet part of any registered landscape; their keys
# (bad = p: <local-factor template> ...) will be added with the conductor>1 work.


# R0R0R0N1: three Gamma_R factors with shift 0 (R0 R0 R0), conductor 1 (N1).
[landscape R0R0R0N1]
description = GL(3,Z) Maass forms, conductor 1, trivial central character
degree      = 3
signature   = 3, 0
conductor   = 1
character   = trivial
self_dual   = false
gamma       = R(0 | 1, 0)
gamma       = R(0 | 0, 1)
gamma       = R(0 | -1, -1)
